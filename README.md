# Expense Logger

A modular Python command-line application to log daily expenses by category, save entries into a JSON file, and produce spending summaries.

## Features
- Add, Edit, Delete, View, Search, and Sort expenses.
- Generate Weekly & Monthly spending summaries.
- Display visual pie charts of spending categories (requires `matplotlib`).
- Backup data functionality.

## How to Run
1. Install dependencies: `pip install matplotlib`
2. Run the program: `python main.py`
